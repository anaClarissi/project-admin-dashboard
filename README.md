## 📌 Sobre o projeto

Este projeto consiste na criação de um painel administrativo (dashboard), com layout moderno e organizado, utilizando principalmente **CSS Grid** para estruturar a interface.

O objetivo foi praticar:

* Estruturação semântica com HTML
* Layouts avançados com CSS Grid
* Organização de componentes visuais
* Estilização com variáveis CSS
* Uso de ícones e fontes externas

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Bootstrap (apenas utilitário)
* Bootstrap Icons
* Google Fonts (Roboto)
